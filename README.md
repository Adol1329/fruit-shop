## Getting Started

1. **Clone the repository**
   ```bash
   git clone [your-repository-url]
   cd project
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   The application should be running at `http://localhost:5173`
